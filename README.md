# Ronak Rajput Portfolio

A Spider-Verse inspired personal portfolio built to showcase my readiness for graduate Software Engineer and Cloud Support Associate roles. The site presents my experience, technical skills, cloud projects, education, certifications, achievements, and contact links in one interactive frontend experience.

## About

This portfolio is designed as a recruiter-facing profile for Ronak Rajput. It highlights practical software engineering and cloud support strengths, including backend development, AWS and Azure fundamentals, troubleshooting, monitoring, logging, CI/CD, distributed systems, and full-stack project delivery.

The experience includes cloud internship work with AWS Lambda and CloudWatch, cloud and VDI migration support, networking and operating system troubleshooting, and software development internship work across frontend and backend layers.

## Key Features

- Interactive Spider-Verse themed UI with animated background effects
- Voice walkthrough assistant with wake word support and manual start option
- Minimizable walkthrough panel that collapses into a circular launcher
- Sections for hero, about video, experience, skills, projects, education, certificates, achievements, and contact
- Infinite certificate marquee showcasing AWS, Azure, and data analytics certificates
- Contact form that opens a pre-filled email draft
- Responsive design for desktop and mobile screens

## Getting Started

### Prerequisites

Install Node.js and npm before running the project locally.

### Installation

```bash
npm install
```

### Run Locally

```bash
npm run dev
```

Open the local URL printed in your terminal, usually:

```text
http://localhost:5173
```

### Build for Production

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```
